# Math Blitz 🎮

**Math Blitz** adalah game matematika cepat bergaya retro arcade dengan visual neon futuristik.

Pemain harus menjawab hasil penjumlahan dua angka secepat mungkin sebelum waktu habis.
Jika hasil penjumlahan lebih dari 9, pemain hanya perlu memasukkan **digit terakhir** dari hasil tersebut.

Contoh:

* `7 + 8 = 15` → jawaban: **5**
* `9 + 9 = 18` → jawaban: **8**

---

## Preview

Game memiliki nuansa:

* Retro arcade UI
* Neon cyber aesthetic
* Countdown transition antar level
* Particle effects saat jawaban benar
* Progress tracking per level

---

## Cara Bermain

1. Tekan tombol **MULAI MAIN**
2. Jawab hasil penjumlahan dua angka
3. Gunakan angka `0-9` di keyboard atau klik numpad di layar
4. Selesaikan sebanyak mungkin soal sebelum waktu habis
5. Terdapat **10 level**
6. Setiap level memiliki:

   * Target: **100 tingkat**
   * Waktu: bertambah **10 detik** per level
